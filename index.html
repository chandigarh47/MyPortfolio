Elemental

Project 3 of the GA WDI course. This is a Ruby On Rails app running at Heroku.

My goal for this project was to present a database driven website, with a clean and simple design, using several javascript libraries and ruby gems. In particular I wanted to manipulate the data from the database using javascript and present it in the form of graphs and animations.

The website stores and displays data of elements in the periodic table, the scientists that discovered or described these elements, and the countries where the major deposits may be found

  • Built in Ruby on Rails
  • Models, Views, Controllers (MVC) application.
  • Image upload using Cloudinary
  • Webscraping using NokoGiri
  • Javascript libraries: Three.JS, underscore, particle.js and jQuery
  • Google geoCharts and bar charts
  • Bootstrap CSS, CSS Grid
  • Maths to animate the spinning Three.JS electrons
  • Live site at Heroku
  • source code at GitHub

FM_Booker

Project 1 of the GA WDI course. This is a Ruby On Rails app running at Heroku.

My goal for this project was to prototype an idea for a real-world application, with different classes of users, to use data from the database to drive CSS, to integrate some search functionality and google maps.

The application concept is an uber for facilities management companies and service technicians. A booker will create a job, and a technician can select the jobs they want to undertake.

  • Built in Ruby on Rails
  • Models, Views, Controllers (MVC) application.
  • User Login function
  • Search using pgSearch
  • Image upload using Cloudinary
  • Google Maps
  • Bootstrap CSS
  • Live site at Heroku
  • source code at GitHub
  • Project ReadMe
